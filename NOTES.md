# Display Full Text in Terminal

To display full text in terminal, pipe the result to `Format-Table -AutoSize`.

```
get-service winrm | ft -AutoSize
```

# Windows Environment Variables
Computername:
```powershell
$env:COMPUTERNAME
```

User Profile:
```powershell
$env:USERPROFILE
```
